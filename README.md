# Cambie game

This is the first cambie game

## how to run
1. 
first get the code somehow. 
you can use git or whatever you pefer. 

2. 
find the folder in terminal. 
Use cd and find where the folder lies.  

3. 

Then write ``` make ```to run.  
the makefile that i wrote uses gcc compiler(on mac).  
change the makefile to so that the compiler and OS. 
will run the game.  

4. 
write start in menu and enjoy!!!
if you dont know commands or what to do
then go to commands.txt, there should be everything you need!
You can also get help in game if you write /help

## Important
please do not change anything in the "gamecore" folder so you wont ruin the game accidently.
if you want to change the game or modify go to "game" folder

### After you quit no progress is saved!!!
